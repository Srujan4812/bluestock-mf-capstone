## 📅 DAY 4 — Fund Performance Analytics (Today)

### Scope & Tasks:
- **Daily Returns**: Calculate `daily_return = nav_t / nav_t-1 - 1` for all 40 schemes.
- **CAGR Calculations**: Calculate 1yr, 3yr, and 5yr CAGR (using max history 4.4 years as 5yr proxy).
- **Ratios**: Annualized Sharpe Ratio (Rf = 6.5%) and Sortino Ratio (using downside std of negative return days only).
- **Alpha & Beta**: Regress daily returns against Nifty 100 benchmark returns using OLS regression.
- **Max Drawdown**: Calculate peak-to-trough worst drop and its peak/trough date ranges.
- **Fund Scorecard (0-100)**: Create composite weighted score from 3yr return rank (30%), Sharpe rank (25%), Alpha rank (20%), expense ratio rank (15%, inverse), and max drawdown rank (10%, inverse). Ranks are percentile ranks.
- **Comparison Chart**: Plot top 5 funds vs Nifty 50 and Nifty 100, and calculate tracking error.

### Key Deliverables:
- `notebooks/Performance_Analytics.ipynb` — notebook populated and executed.
- `scripts/performance_analytics.py` — core calculation code.
- `scripts/generate_notebook.py` — notebook creator.
- `data/processed/fund_scorecard.csv` — rankings list.
- `data/processed/alpha_beta.csv` — regression results.
- `reports/charts/benchmark_comparison.png` — top 5 vs benchmarks performance chart.
- `reports/day4_performance_report.txt` — return stats, scorecard list, tracking errors.
- Git commit: "Day 4: Completed Fund Performance Analytics, composite scorecard, and charts"
